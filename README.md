# Customer Shopping Behavior Analysis

## Overview
This project showcases an end-to-end data analytics workflow using Python, SQL, and Power BI. It includes dataset loading, EDA, data cleaning, SQL analysis, dashboard building, and a final presentation created with Gamma.

## Dataset
- Customer Shopping Behavior dataset (CSV)
- Contains demographics, purchase details, transaction amounts, and product categories.

## Tools Used
- **Python:** Pandas, NumPy, Matplotlib/Seaborn  
- **Databases:** PostgreSQL / MySQL / SQL Server  
- **Visualization:** Power BI  
- **Presentation:** Gamma (PPT)

## Steps
1. **Load Data:** Imported CSV and reviewed structure.  
2. **EDA:** Explored trends, distributions, and correlations.  
3. **Clean Data:** Handled missing values, duplicates, and formatting.  
4. **SQL Queries:** Analyzed sales, customers, and category performance.  
5. **Dashboard:** Built an interactive Power BI dashboard with KPIs and filters.  
6. **Report & PPT:** Summarized insights through a Power BI report and Gamma presentation.

## Dashboard
Interactive Power BI dashboard showing sales trends, customer insights, and product category performance.

## Results
- Identified top-selling categories  
- Highlighted peak shopping periods  
- Found high-value customer groups  
- Extracted key insights to support business decisions

## How to Run
1. Clone the repository  
2. Install requirements: `pip install -r requirements.txt`  
3. Open and run the Jupyter Notebook  
4. Load cleaned data into SQL and run queries  
5. Open the Power BI `.pbix` file  
6. View the Gamma PPT in the `presentation` folder
